## 实验建议

!!! tip "完成实验的建议"
    - **先理解后动手**: 认真阅读任务书，做好设计，再动手写代码
    - **分步实现**: 将大任务分解为小步骤，逐步完成并测试
    - **善用调试工具**: 熟练使用 GDB 在 QEMU 中调试，能大幅提高效率
    - **代码规范**: 保持良好的代码风格，添加必要的注释
    - **及时提问**: 遇到问题及时向助教或同学请教

!!! warning "注意事项"
    - 每个实验的 S core 都是后续实验的基础，务必扎实完成
    - 独立思考和完成，禁止抄袭
    - 注意截止时间，合理安排时间
