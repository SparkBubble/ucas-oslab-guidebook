## 学习路径

建议按照以下顺序进行学习：

1. **准备阶段** (P0): 完成环境搭建，学习必备基础知识
2. **启动阶段** (P1): 理解系统启动，掌握镜像制作
3. **内核阶段** (P2): 实现进程管理和基本调度
4. **完善阶段** (P3): 添加进程通信和多核支持

## 实验建议

!!! tip "完成实验的建议"
    - **先理解后动手**: 认真阅读任务书，理解原理后再开始编码
    - **分步实现**: 将大任务分解为小步骤，逐步完成并测试
    - **善用调试工具**: 熟练使用 GDB 和 QEMU 调试，能大幅提高效率
    - **代码规范**: 保持良好的代码风格，添加必要的注释
    - **及时提问**: 遇到问题及时向助教或同学请教

!!! warning "注意事项"
    - 每个实验都是后续实验的基础，务必扎实完成
    - 独立思考和完成，禁止抄袭
    - 注意提交截止时间，合理安排时间
