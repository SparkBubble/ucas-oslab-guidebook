# 欢迎来到 UCAS 操作系统研讨课

!!! info "课程简介"
    本课程是中国科学院大学（UCAS）计算机科学与技术学院的操作系统研讨课程。通过一系列实验任务，学生将深入理解操作系统的核心概念和实现原理，动手设计、亲身实践，从头实现一个自己的操作系统，而不是拘泥于理论课上的“纸上谈兵”。

## 课程目标

通过该实验课，大家将由浅入深，从操作系统的引导到操作系统复杂的内部实现，一步一步，深刻理解操作系统中进程的管理、中断的处理、内存的管理、文件系统等相关知识，并将其实现出来。我们衷心希望，每个选修完这门课的同学以后都能自豪地说“我自己实现过一个完整的操作系统”。

## 实验内容

本课程目前已经发布如下实验任务：

| 实验编号 | 实验名称 | 主要内容 |
|---------|---------|---------|
| P0 | 准备知识 | 开发环境、工具链、RISC-V 基础 |
| P1 | 引导、镜像文件和ELF文件 | 系统引导、镜像制作、ELF 文件结构 |
| P2 | 简易内核实现 | 进程管理、调度、系统调用 |
| P3 | 进程管理、通信与多核执行 | Shell、进程通信、多核调度 |
