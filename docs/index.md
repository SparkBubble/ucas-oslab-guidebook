# 欢迎来到 UCAS 操作系统研讨课

!!! info "课程简介"
    本课程是中国科学院大学（UCAS）计算机科学与技术学院的操作系统研讨课程。通过一系列实验任务，学生将深入理解操作系统的核心概念和实现原理，动手设计、亲身实践，从头实现一个自己的操作系统，而不是拘泥于理论课上的“纸上谈兵”。

## 课程目标

通过该实验课，大家将由浅入深，从操作系统的引导到操作系统复杂的内部实现，一步一步，深刻理解操作系统中进程的管理、中断的处理、内存的管理、文件系统等相关知识，并将其实现出来。我们衷心希望，每个选修完这门课的同学以后都能自豪地说“我自己实现过一个完整的操作系统”。

## 实验内容

本课程包含以下实验任务：

| 实验编号 | 实验名称 | 主要内容 |
|---------|---------|---------|
| Project 1 | 引导与中断 | 系统引导流程、中断处理机制 |
| Project 2 | 物理内存管理 | 物理内存分配与管理 |
| Project 3 | 虚拟内存管理 | 分页机制、虚拟地址转换 |
| Project 4 | 进程管理 | 进程创建、调度与同步 |
| Project 5 | 文件系统 | 文件系统设计与实现 |

## 学习建议

!!! tip "学习技巧"
    - 按顺序完成实验，每个实验都是后续实验的基础
    - 先做设计，充分理解原理后再动手写代码
    - 多使用调试工具，培养系统级调试能力
    - 有困难及时向助教或者其他同学请教

!!! warning "注意事项"
    - 尽量尝试独立完成实验，培养独立思考和解决问题的能力
    - 遵守学术诚信，禁止抄袭
    - 在指定时间内提交代码并验收
